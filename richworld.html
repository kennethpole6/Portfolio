Human Resource Information System Roles: UI/UX Designer & Front-End Developer

Company: Richworld Giftshop

Human Resource Information System for Richworld Giftshop is a software solutions for the company to organize their employee information. It comes with handling big data information with security. We also developed job listing all the way to the hiring processes of an employee. As for the user, he/she can browse the listed jobs from the website directly. The user can also submit their resumes if they find a job that interest them.

  • Features
  • • Easily organize the data/information of employee.
  • • The user can find a job listing he desires with description/details in the app.
  • • The user will be notified if the applicant is suitable for a job.
  • • The user can submit job requirements (resumes/CV) directly in website.
  • • The system has an built-in messages to communicate simultaneously and seamlessly.
