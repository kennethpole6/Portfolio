Swab Appointing System Roles: UI/UX Designer


Swab Appointing System is a web based scheduling and appointing of swab tests. The user can easily appoint from the web and then the system will schedule the available time and date for both user and Doctor. In this case, this will eliminate the long waiting of queue just to get an appointment. After the process, the user will see the results when the process are finished.

  • Features
  • • Easily organize the appointing and scheduling of the user.
  • • The user can appoint and schedule straight from the web so that he will never be waiting in long queue.
  • • The results will be seen from the user side after the process is finished within a days or week.
